  3. Find all moves for a certain pokemon, in this case, Rowlet (Three Table Join)
    Query: SELECT p.pokeName, m.moveName FROM moves m, moveLearnset s, pokemon p WHERE m.moveID = s.moveID AND s.pokeDexID = p.pokeDexID AND p.pokeName = 'Rowlet'; Result:
    4. pokeNamemoveName
    RowletAstonish
    RowletGrowl
    RowletLeafage
    RowletTackle
  4. Find all locations that spawn more than one pokemon. (Aggregate Function with GROUP BY and HAVING)
    Query: SELECT COUNT(sp.locationID) as NumOfPokemon, l.locationName FROM spawnsPokemon sp, locations l WHERE sp.locationID = l.locationID group by locationName having COUNT(sp.locationID) > 1;

  6. Find the number of items that are spwaned in locations (Aggregate Function)
    Query: SELECT COUNT(si.locationID) AS NumOfItems, l.locationName AS Location FROM spawnsItem si, locations l WHERE l.locationID = si.locationID GROUP BY locationName;
